# PDF 内容提取技能

## 功能描述

根据用户提供的 PDF 文件路径，自动提取 PDF 文件中的文本内容并输出到控制台。

## 使用方式

用户输入格式：
```
提取 PDF: [文件路径]
```

或

```
读取 PDF 内容：[文件路径]
```

示例：
- "提取 PDF: D:\documents\report.pdf"
- "读取 PDF 内容：C:\Users\Public\test.pdf"z

## 执行流程

1. **解析用户输入** - 提取 PDF 文件路径
2. **验证文件存在** - 检查文件路径是否有效且为 PDF 文件
3. **读取 PDF 内容** - 使用 PyMuPDF (fitz) 库提取文本
4. **输出到控制台** - 将提取的文本内容打印显示

## 依赖要求

- Python 3.7+
- PyMuPDF 库 (`pip install PyMuPDF`)
- Windows/Linux/macOS 系统

## 注意事项

- 支持绝对路径和相对路径
- 自动处理文件路径中的反斜杠和正斜杠
- 如果 PDF 有密码保护，会提示用户输入密码
- 扫描版 PDF（图片格式）无法提取文本，会给出提示
- 大文件可能会输出较多内容，建议分页显示

## 参数说明

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| file_path | string | 是 | PDF 文件的完整路径 |
| password | string | 否 | 加密 PDF 的密码（可选） |

## 输出格式

```
========================================
PDF 文件：[文件名]
总页数：[页数]
========================================

[第 1 页内容]
...

[第 2 页内容]
...

========================================
提取完成
========================================
```
