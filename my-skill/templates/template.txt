# 微信文件发送请求模板

## 请求格式

```
发送文件给 [用户名]: [文件名]
```

## 可替换参数

- `[用户名]` - 微信联系人的姓名、昵称或备注名
- `[文件名]` - 桌面上的文件名（需包含扩展名）

## 示例

```
发送文件给 张三：report.pdf
```

```
把 工作总结.docx 发给 李经理
```

```
发送 预算表.xlsx 给 王财务
```

## 快捷短语

- "发文件给 {用户名}: {文件名}"
- "把 {文件名} 发给 {用户名}"
- "微信发 {文件名} 给 {用户名}"
- "{文件名} 发给 {用户名}"
